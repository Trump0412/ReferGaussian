Benchmark

R4D-Bench-QA

12 dynamic scenes, 266 query annotations, and four query categories with frame-aligned supervision.

Figure 2. R4D-Bench-QA statistics: expression word cloud, query composition, and token-length distribution.

Method

HyperGaussian Framework

Dynamic reconstruction plus a Qwen-based Hyper-Planner in three explicit referring stages.

Figure 3. Pipeline overview: dynamic reconstruction, static segmentation, semantic assignment, and spatio-temporal reasoning.

Stage 1

Static Segmentation

Grounded-SAM2 tracks candidate entities across time and provides mask evidence for Gaussian-to-entity association.

Stage 2

Semantic Assignment

Entity-level visual and semantic descriptors are written into a reusable EntityBank shared by all queries in a scene.

Stage 3

Spatio-Temporal Reasoning

The Hyper-Planner decomposes language constraints and selects final referents from EntityBank without scene-specific retraining.
